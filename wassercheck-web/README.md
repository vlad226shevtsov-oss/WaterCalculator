
# WasserCheck Web

Отдельное браузерное приложение для расчёта расхода воды, энергии и стоимости.
Дизайн основан на предоставленном макете WasserCheck.

## Запуск

```bash
pnpm install
pnpm dev
```

После запуска откройте адрес, показанный Vite (обычно `http://localhost:5173`).

## Возможности

- расчёт для душа и ванны;
- стоимость воды и нагрева;
- прогноз на месяц и год;
- немецкий и русский языки;
- EUR, USD и UAH;
- адаптивный интерфейс для компьютера и телефона.
